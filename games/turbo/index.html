Lives: ❤️ ❤️ ❤️

Turbo needs your help to get safely to the finish flag!

However, there are hidden monsters in the minefield that want to eat Turbo!
If you encounter a monster, you lose a life.

There is exactly one monster in each row, except the top and bottom rows, which are safe.

There is at most one monster in each column.

Good luck!

Oh, and something you can ponder on: How many lives does Turbo need to be guaranteed
to make it across when there are N columns? Maybe 3 lives is not always enough?
